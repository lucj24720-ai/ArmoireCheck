LES 5 ETAPES (VERSION ULTRA-COURTE):
================================================================================

    1. Aller sur https://neon.tech et creer un compte (2 min)

    2. Copier la "connection string" (30 sec)
       (ca ressemble a: postgres://user:pass@ep-xxx.neon.tech/db)

    3. Ouvrir le fichier backend/.env (30 sec)
       Remplacer la ligne 3 avec votre connection string

    4. Dans Neon, aller dans SQL Editor (1 min)
       Executer init.sql puis seed.sql

    5. Demarrer l'app (1 min)
       Terminal 1: cd backend && npm run dev
       Terminal 2: cd frontend && npm start

    TERMINE ! Ouvrir http://localhost:3000

ERREURS COURANTES ET SOLUTIONS:
================================================================================

    Erreur: "ECONNREFUSED"
    Solution: La base de donnees n'est pas connectee
              -> Suivez DEMARRAGE_RAPIDE.md

    Erreur: "password authentication failed"
    Solution: Mauvaise connection string
              -> Recopiez la connection string complete depuis Neon

    Erreur: "relation does not exist"
    Solution: Les tables n'ont pas ete creees
              -> Executez init.sql et seed.sql dans Neon SQL Editor

    Erreur: "Port 5001 already in use"
    Solution: Un autre processus utilise le port
              -> Voir commandes dans SOLUTION_RAPIDE.md
